Mobile App Development ✨

Hawa

Islamic Personal & Community Hub ★

Overview ★

Hawa is an innovative mobile application designed to create a holistic digital space for Muslims to track their spiritual journey, connect with community, and access personalized Islamic resources. As both UX/UI Designer and Developer, I crafted an intuitive and aesthetically pleasing platform that bridges technology with spiritual growth.

Key Technologies ★
  • React Native
  • Firebase
  • Figma
  • UX/UI Design
  • Mobile Development
  • Community Platform
Key Features ✨
  • Customizable user profiles
  • Spiritual goal tracking dashboard
  • Interactive learning sections
  • Community connection features
  • Personalized resource recommendations
Design Philosophy ★

The design of Hawa was driven by a commitment to creating an inclusive, intuitive, and spiritually enriching digital experience. I focused on developing an interface that feels both modern and respectful, using a color palette and design language that resonates with the app's spiritual purpose.
